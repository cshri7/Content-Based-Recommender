The break keyword exits a flow control loop, such as a for loop. It basically breaks the loop. In the code section 3.34, the loop would print out all the numbers from 1 to 10, but we have a check for when i equals 5. When the loop reaches its fifth iteration, it will be cut short by the break statement, at which point it will exit the loop. Code section 3.34: An interrupted for loop.1 for (int i = 1; i <= 10; i++) {2     System.out.println(i);3     if (i == 5) {4        System.out.println("STOP!");5        break;6     }7 }Output for code section 3.3412345STOP!
